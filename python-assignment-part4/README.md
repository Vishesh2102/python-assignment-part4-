# Student Performance Analysis & ML Prediction
An end-to-end data science project using Pandas, Matplotlib, Seaborn, and Scikit-Learn.

### Workflow:
1. **Exploration:** Summarizing student data using Pandas `.describe()` and `.mean()`.
2. **Visualization:** - Matplotlib for distribution (Histograms/Boxplots) and trends (Line/Scatter).
   - Seaborn for statistical comparison and regression lines.
3. **Machine Learning:** - Feature scaling with `StandardScaler`.
   - Training a `LogisticRegression` classifier to predict Pass/Fail outcomes.
   - Analysis of feature coefficients to determine which factors (like study hours) most impact success.
